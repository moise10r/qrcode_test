## QR CODE TEST

**Cloning the Repository**

```bash
git clone https://github.com/moise10r/qrcode_test
cd qrcode_test
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Running the Project**

```bash
npx  tsc
```

```bash
node qrcode.js
```

**Running test**

```bash
npx jest
```
